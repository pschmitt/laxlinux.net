Barre de recherche

a Blork search engine v0.3

Blork est un script php largement personnalisable qui permet de mettre en place une barre de recherche, sans être contraint d'utiliser une base de données MySQL.

Nota Bene: ce script, distribué sous une license GPL v2, n'est plus en développement depuis 2008. Il est donc nécessaire d'y entreprendre quelques modifications, comme transformer les ereg_replace par des preg_replace ou modifier la fonction 'explore' qui ne parcourt pas tous les fichiers avec PHP≥5.

Script JS

Les scripts permettant d'effacer le contenu par défaut de la barre de recherche lorsque l'on clique dessus sont issus de yourhtmlsource.com .
